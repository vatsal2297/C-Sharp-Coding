# This is the solution of BeeBreeding Problem where we need to find the distance between two cells in a hexagonal coordinate system.

The solution conists of the following files and/or folders:
* _BeeBreedingProblem_
* _BeeBreedingProblem.sln_

## NOTE: This file is for Visual Studio, so after downloading the solution, open the .sln file and it will open in visual studio.

__Running the program and getting output:__

1) Enter two numbers(positive) in two different lines.
2) Output will be a number with a prompt showing the distance between two given cells/numbers.
